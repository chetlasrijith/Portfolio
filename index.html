Selected Works.

Eco Bot

Machine Learning Project

Eco Bot is an AI-powered solution designed to tackle underwater plastic pollution. Using a YOLOv8 object detection model, it identifies and locates plastic waste in underwater images or live video feeds with high accuracy. What makes Eco Bot unique is its integration with GPS data, enabling real-time mapping of pollution hotspots. Detected locations are visualized on an interactive map using Leaflet.js, helping environmental researchers and cleanup teams prioritize areas with high plastic density. Project Link.
